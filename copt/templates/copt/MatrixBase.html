MatrixBase

MatrixBase is one of the base mathematical classes in open source library COPT. MatrixBase describes a basic type matrix. A matrix is a multiple dimensional array with a number of arithmetic operations including summation, subtraction, multiplication etc. This class derives directly from another basic type called Array and also takes scalar and index as template.

MatrixBase()
The default constructor of MatrixBase. A matrix with 0 row and 0 column will be generated as default.
MatrixBase(const index m, const index n, const scalar *data=NULL)

Parameters:

m: Row number of the matrix

n: Colunm number of the matrix

data: The pointer to the array of data. NULL is used as default.

scalar& operator()(const index i, const index j)
Matlib-like data access of MatrixBase. User can use operator() to access the element at the (i+1)-th row and the (j+1)-th colunm of a given MatrixBase mat. The syntax is simply mat(i,j). Note that the first element is with index (0,0) meaning that i is in range [0,rows-1] and j is in range [0,colunms-1].

Parameters:

j: The column index number of the element we want to get

Returns:

The (i+1,j+1) element of the matrix
