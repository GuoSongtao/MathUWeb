VectorBase

VectorBase is one of the base mathematical classes in open source library COPT. VectorBase describes a basic type vector. A vector is a multiple dimensional array with a number of arithmetic operations including summation, subtraction, multiplication etc. This class derives directly from another basic type called Array and also takes scalar and index as template.

VectorBase()
The default constructor of VectorBase. A zero-dimensional vector will be generated as default.
VectorBase(const index size, scalar *data = nullptr)
Constructor for generating a vector with specific length and a given array. A zero vector is generated if given array is a nullptr.

Parameters:

size: The size of the vector

data: The pointer to the array of data. nullptr is used as default.

VectorBase ( const VectorBase& vec )
Copy constructor from a given vector vec. The vector will still be a reffered vector if vec is one. Both vectors point to the same array. Otherwise the data will be copied.
VectorBase( const index size, const referred_array& tag, scalar* data, const index inter = 1 )
Constructor of a referred vector. Referred vector is a special kind of vector which does not have its own data. The data is stored somewhere else. As an instance, it is not convenient to construct another vector when representating a column or a row of a matrix. But it is efficient just referred to the data. Here inter is the interval between two elements.
VectorBase( const index size, const referred_array& tag, const scalar* data, const index inter = 1 )
The same as above constructor.
VectorBase(const std::vector& vec)
Constructor from std::vector vec from C++ standard libraray. The size is set as the size of std::vector and the data is the same as given vec.
~VectorBase()
The deconstructor of VectorBase.
VectorBase& operator=( const VectorBase& vec )
Copy assignment of VectorBase. The functionality is the same as copy constructor. All data is copied.
scalar& operator()(const index i )
Matlib-like data access of VectorBase. User can use operator() to access the i-th element of a given VectorBase vec. The syntax is simply vec(i) and out-of-range exception will be throwed if i is less than zero or no less than the size of vec. Note that the first element is with index 0 meaning that i is in range [0,size-1].
const scalar& operator() (const index i )const
Const access of element in VectorBase.
bool operator< (const VectorBase& vec)const
Comparison of two VectorBase. Value true is returned if every element of the first VectorBase is smaller than the right one, false is returned otherwise.
bool operator< (const scalar s)const
Comparison of a VectorBase and a scalar. Value true is returned if every element of the VectorBase is smaller than the given scalar s, false is returned otherwise.
bool operator<=(const VectorBase& vec)const
This is operator<= of VectorBase.Return true if every element of the left VectorBase is not larger than corresponding element of the right one, return false otherwise.
bool operator<=(const scalar s)const
This is operator<= of VectorBase.Return true if every element of the left VectorBase is not larger than the given scalar s, return false otherwise.
bool operator> (const VectorBase& vec)const
This is operator> of VectorBase.Return true if every element of the left VectorBase is larger than corresponding element of the right one, return false otherwise.
bool operator> (const scalar s)const
This is operator> of VectorBase.Return true if every element of the left VectorBase is larger than the given scalar s, return false otherwise.
bool operator>=(const VectorBase& vec)const
This is operator>= of VectorBase.Return true if every element of the left VectorBase is not smaller than corresponding element of the right one, return false otherwise.
bool operator>=(const scalar s)const
This is operator>= of VectorBase.Return true if every element of the left VectorBase is not smaller than the given scalar s, return false otherwise.
bool operator==(const VectorBase& vec)const
This is operator== of VectorBase.Return true if every element of the left VectorBase is equal to corresponding element of the right one, return false otherwise.
bool operator==(const scalar s)const
This is operator== of VectorBase.Return true if every element of the left VectorBase is equal to the given scalar s, return false otherwise.
bool operator!=(const VectorBase& vec)const
This is operator!= of VectorBase.Return true if at least one element of the VectorBase is not equal to corresponding element of the right one, return false otherwise.
bool operator!=(const scalar s)const
This is operator!= of VectorBase.Return true if at least one element of the left VectorBase is not equal to the given scalar s, return false otherwise.
podscalar squaredNorm() const
Computation of the square of l2 norm of the VectorBase.
podscalar absNorm() const
Computation of the abs norm or l1 norm of the VectorBase.
podscalar normalize()
Normalizing the VectorBase and returning the norm of it.
scalar dot(const VectorBase& vec) const
This is the dot operation of VectorBase.
void scale(scalar s)
Scaling the whole vector with a given scalar s.
VectorBase operator* (scalar s)
Computing the multiplication of the vector and a scalar s. Every element of the vector multiplies s.
friend VectorBase operator* (scalar s,const VectorBase& vec)
This is the same function as "VectorBase operator* (scalar s)".
VectorBase operator+ (const VectorBase& vec) const
This is the summation operation of VectorBase.
VectorBase operator- (const VectorBase& vec) const
This is the subtraction operation of VectorBase.
VectorBase operator- () const
This is a function used to calculate Anti vector.
friend std::ostream& operator<<(std::ostream& os,const VectorBase& vec)
Overload of output stream of VectorBase. The output form is like [v[0],v[1],...,v[size-1]].
static VectorBase vecE(index size,index i)
This function generates a VectorBase with specific size whose i-th element is 1 while others are 0.
static VectorBase vecE(index size,index i,const scalar s)
This function generates a VectorBase with specific size whose i-th element is s while others are 0.
MatrixBase mulTrans(const VectorBase& vec) const
This is the transpose operation of VectorBase.
VectorBase transMul(const MatrixBase& mat) const
This function transposes a VectorBase and multiplies it with a Matrix.
VectorBase block(const std::set<index>& indices) const
Vector blocking of the vector. The indices of blocking vector are given in std::set<index> from C++ standard library. Notice that the elements of std::set<index> are placed in order.
VectorBase block(const std::vector<index>& indices) const
Vector blocking of the vector. The indices are given std::vector<index> from C++ standard library. Remember that the elements of std::vector<index> are not forced to be in order.
template<class InputIterator>
VectorBase block(const index bs, InputIterator begin, InputIterator end) const
Another vector blocking method taking iterator as input. The begin iterator and end iterator are used as parameters.
void blockFromVector(const VectorBase& vec, const std::set<index>& indices)
Blocking the vector from a given vector vec. The indices are given in std::set<index> from C++ standard library.
void blockFromVector(const VectorBase& vec, const std::vector<index>& indices)
Blocking the vector from a given vector vec. The indices are given in std::vector<index> from C++ standard library.
template<class InputIterator>
void blockFromVector(const VectorBase& vec, const index bs, InputIterator begin, InputIterator end)
Blocking the vector from a given vector vec. The begin and end iterator are given.
void combine( const VectorBase& v1, const VectorBase& v2)
This is a function used to combine two vectors.
static inline void stCombine( const VectorBase& v1, const VectorBase& v2, VectorBase& v)
This is a function used to combine two vectors and it takes output as a parameter.
static inline VectorBase random( const index i )
This function generates a random VectorBase with size i.
